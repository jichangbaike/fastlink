## 线路倍率说明

FastLink 当前主要线路倍率结构如下：

- **BGP 动态专线：** 3 倍率  
- **IPLC 专线：** 1.5 倍率  
- **福利节点：** 0.1 倍率（极低倍率，适合轻量使用）

> 倍率表示流量消耗系数。倍率越高，相同使用量消耗的套餐流量越多。
